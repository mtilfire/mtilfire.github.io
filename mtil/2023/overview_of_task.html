Machine Translation for Indian Languages

MTIL (2023)


The participants of this program will have access to a comprehensive set of resources including development, training, and test datasets. There are two main tasks in this program.

General Translation Task(Task 1) Task participants should build a machine translation model to translate sentences of the following language pairs:

  1. Hindi-Gujarati
  2. Hindi-Kannada
  3. Kannada-Hindi
  4. Hindi-Odia
  5. Odia-Hindi
  6. Hindi-Punjabi
  7. Punjabi-Hindi
  8. Hindi-Sindhi
  9. Urdu-Kashmiri
  10. Telugu-Hindi
  11. Hindi-Telugu
  12. Urdu-Hindi
  13. Hindi-Urdu

Domain Specific Translation Task(Task 2) Task participants will build machine translation models for Governance and Healthcare domains. The language pairs for each domain are as follows:

  1. Healthcare:
    • Hindi-Gujarati
    • Kannada-Hindi
    • Hindi-Odia
    • Odia-Hindi
    • Hindi-Punjabi
    • Kannada-Hindi
  2. Governance:
    • Hindi-Gujarati
    • Kannada-Hindi
    • Hindi-Odia
    • Odia-Hindi
    • Hindi-Punjabi
    • Kannada-Hindi

By participating in these tasks, participants will have the opportunity to showcase their expertise in machine translation, contribute to the advancement of translation technology, and tackle the difficulties associated with domain-specific translation problems.
